复杂系统

知识点：

  1. 用转向力计算公式(转向力 = 所需速度 - 当前速度) 实现群集的规则；
  2. 这些转向力有群体产生， 小车要根据所有其他小车的状态计算转向力；
  3. 我们需要结合多个转向力，并对他们进行加权；
  4. 模拟结果是一个复杂系统——智能的群体行为将从简单的群及规则中产生，系统中没有控制中心和领导者
